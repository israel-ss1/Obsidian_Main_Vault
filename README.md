# Obsidian Main Vault 

Estes é o repositório pessoal de Israel Silva de Souza. Os estudos e escritos são de propriedade intelectual do autor. Trata-se de reflexões, estudos, ideais e pensamentos segundo sua cosmovisão. 

A maioria dos estudos é sobre teologia, filosofia, história, literatura e áreas relacionadas. Fazem parte da estrutura de arquivos markdown, utilizado pelo software Obsidian, criado e popularizado após a pandemia de COVID-19

Se tiver interesse em scripts e templates, entre em contato comigo para aquisição em outro repositório, uma vez que este está em constante teste e aperfeiçoamento.
